# PPT Generator

Use this skill to create complete, editable presentation deliverables.

## When to Invoke

Invoke this skill when the user asks for:

- PPT or PPTX generation
- Slide deck drafting
- Pitch deck creation
- Training/report presentation output
- Converting structured content into slides

## Workflow

1. Confirm presentation objective, audience, and language.
2. Define slide count and structure (title, agenda, body, summary, Q&A).
3. Draft slide-by-slide content with clear titles and concise bullet points.
4. Specify visual guidance per slide (chart, table, diagram, image placeholder).
5. Generate an editable PPTX file and ensure file naming is consistent with topic/version.
6. Validate content consistency, tone, and formatting before delivery.

## Output Requirements

- Provide a slide outline first when scope is unclear.
- Keep one core message per slide.
- Prefer short bullets over long paragraphs.
- Ensure consistent terminology across all slides.
- Include suggested speaker notes when user requests them.

## Quality Checklist

- Objective aligned with user intent
- Logical story flow across slides
- Readable text density
- Actionable conclusion slide
- Exported file is editable and complete
