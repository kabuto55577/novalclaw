# 金融分析（财务报表与经营质量）

用于对上市公司或非上市主体的**财务报表与经营质量**进行结构化分析；不替代持牌投资顾问意见，需提示用户自主决策与合规边界。

## 何时启用

在以下场景主动运用本技能：

- 解读利润表、资产负债表、现金流量表及附注要点
- 计算并解释 **盈利能力、成长性、运营效率、偿债能力、现金创造** 等比率
- 同业对比、历史趋势、结构性变化（收入确认、费用、一次性项目）
- 识别红旗：应收/存货异常、商誉、关联交易、资本化 vs 费用化、现金流与利润背离

## 分析流程

1. **明确标的与时间范围**：报告期、币种、会计准则（CAS / IFRS / US GAAP）、是否并表。
2. **三张表联动**：利润 → 资产负债变动 → 现金流验证；优先看 **经营现金流** 与净利润匹配度。
3. **比率套件**（按场景选取，给出公式与口径）：
   - 盈利：毛利率、营业利润率、净利率、ROE、ROA、ROIC
   - 成长：营收/毛利/净利 YoY、环比；同店/分部增速（若披露）
   - 效率：资产周转、存货/应收周转天数、现金循环周期
   - 杠杆与偿债：资产负债率、有息负债、利息覆盖、流动/速动比率
   - 现金：FCF、FCF 转化率、资本开支与折旧关系
4. **质量与风险**：盈利可持续性、一次性项目剥离后利润、分部披露、前五大客户/供应商集中度。
5. **输出结构**：执行摘要 → 关键数据表 → 比率解读 → 主要假设与局限 → 需进一步核实的问题。

## 输出要求

- 数字须注明**单位与报告期**；同比/环比说明清楚。
- 对缺失数据明确写「未披露」或「无法从给定材料推断」，避免臆测。
- 涉及投资建议时，使用**情景分析**与**风险提示**，避免单一目标价承诺。
- 中国大陆市场可提示：业绩预告/快报规则、ST 风险、分红与再融资约束等**一般性**合规提示（非法律意见）。

## 质量检查清单

- 口径一致（非经常性损益是否剔除、是否扣非）
- 现金流与利润逻辑自洽
- 同业对比使用可比口径
- 重要假设已列出敏感性
