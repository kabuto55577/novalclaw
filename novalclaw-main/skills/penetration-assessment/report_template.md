## 2. 执行摘要

- **总体结论**：通过 / 不通过 / 有条件通过  
- **最高风险项**：1～3 条一句话描述  
- **业务影响概述**：机密性 / 完整性 / 可用性  

## 3. 测试范围与方法

- **范围内资产**：IP、域名、应用、API  
- **范围外说明**：未测试项及原因  
- **方法**：自动化扫描 + 手工验证 +（可选）代码审计引用  

## 4. 信息收集与攻击面

## 5. 发现与漏洞详情

| 字段 | 说明 |
|------|------|
| ID | 如 VULN-001 |
| 标题 | 简短名称 |
| 严重程度 | 严重 / 高 / 中 / 低 / 信息 |
| 影响 | 被利用后的最坏情况 |
| 位置 | URL、参数、组件版本 |
| 复现步骤 | 分步、可交给研发复现 |
| 证据 | 截图、请求/响应摘要（脱敏） |
| 修复建议 | 具体配置或补丁方向 |
| 参考 | CVE、CWE、厂商公告 |
